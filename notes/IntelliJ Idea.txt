SHORTCUTS
=========

- New Java Class: CTRL + ALT + INS
- Add Constructor Param: ALT + ENTER
- Complete Statement: SHIFT + CTRL + ENTER
- Extract to Variable: CTRL + ALT + V
- Search Anything: 2 x SHIFT
- Go to Test: CTRL + SHIFT + TEST
